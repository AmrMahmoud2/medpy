Installing MedPy in development modeÂ¶

If you care to work on the source directly, you can install MedPy in development mode. Then the sources will remain and any changes made them them be directly available system-wide. First follow all steps described in Installing MedPy as root except the last, then download the MedPy sources from https://pypi.python.org/pypi/MedPy/, unpack them, enter the directory and run:

sudo python setup.py develop
