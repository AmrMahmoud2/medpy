medpy.itkvtk.utilities.itku.getImageTypeFromArray¶

medpy.itkvtk.utilities.itku.getImageTypeFromArray(arr)[source]¶

Returns the image type of the supplied array as itk.Image template.

Parameters:

arr : array_like

The image as numpy array.
Returns:

image_type : itk.Image (template)

An itk image type.
Raises:

DependencyError

If the itk wrapper do not support the target image type

ImageTypeError

If the array dtype is unsupported
