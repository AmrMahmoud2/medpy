medpy.itkvtk.utilities.itku.getImageFromArray¶

medpy.itkvtk.utilities.itku.getImageFromArray(arr, image_type=False)[source]¶

Returns an itk Image created from the supplied numpy ndarray. If the image_type is supported, will be automatically transformed to that type, otherwise the most suitable is selected.

Parameters:

arr : array_like

The image as numpy array.

image_type : itk.Image (template)

An itk image type.
Returns:

itk_image : itk.Image (instance)

An instance of itk.Image holding the array’s data.

Notes

Always use this instead of directly the itk.PyBuffer, as that object transposes the image axes.
